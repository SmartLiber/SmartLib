## 开发人员 A：用户操作类（UserService）

### 类名：`UserService`

### 负责函数清单：

| 序号 | 函数名 | 功能 | 输入参数 | 返回值 |
|------|--------|------|----------|--------|
| 1 | registerUser | 用户注册 | username, phone | boolean |
| 2 | loginUser | 用户登录 | username | User |
| 3 | getUserBorrowHistory | 查询用户借阅记录 | userId | List<Book> |
| 4 | hasUnreturnedBooks | 验证用户是否有未归还书籍 | userId | boolean |
| 5 | getTotalUserCount | 获取用户总数 | - | int |
| 6 | updateUserInfo | 更新用户信息 | userId, username, phone | boolean |
| 7 | getUserById | 根据ID获取用户 | userId | User |
| 8 | getAllUsers | 获取所有用户 | - | List<User> |

### 函数详细说明：

#### 1. registerUser - 用户注册
```
功能：注册新用户，验证用户名唯一性
输入：username (String), phone (String)
返回：boolean
涉及操作：
  - UserDAO.getUserByUsername() - 检查用户名是否存在
  - UserDAO.addUser() - 添加用户
业务规则：
  - 用户名不能为空且不能重复
  - 手机号格式校验（11位数字）
```

#### 2. loginUser - 用户登录
```
功能：用户登录验证
输入：username (String)
返回：User
涉及操作：
  - UserDAO.getUserByUsername()
业务规则：
  - 用户名存在即登录成功
```

#### 3. getUserBorrowHistory - 查询借阅记录
```
功能：查询用户的借阅历史
输入：userId (int)
返回：List<Book>
涉及操作：
  - UserDAO.getBorrowedBooksByUser()
```

#### 4. hasUnreturnedBooks - 检查未还书籍
```
功能：检查用户是否有未归还的书籍
输入：userId (int)
返回：boolean
涉及操作：
  - UserDAO.getBorrowedBooksByUser()
业务规则：
  - 返回列表非空表示有未归还书籍
```

#### 5. getTotalUserCount - 获取用户总数
```
功能：获取注册用户总数
输入：无
返回：int
涉及操作：
  - UserDAO.getAllUsers()
```

#### 6. updateUserInfo - 更新用户信息
```
功能：更新用户信息
输入：userId (int), username (String), phone (String)
返回：boolean
涉及操作：
  - UserDAO.getUserById()
  - UserDAO.updateUser()
业务规则：
  - 用户必须存在
  - 用户名不能与其他用户重复
```

#### 7. getUserById - 获取用户
```
功能：根据ID获取用户
输入：userId (int)
返回：User
涉及操作：
  - UserDAO.getUserById()
```

#### 8. getAllUsers - 获取所有用户
```
功能：获取所有用户列表
输入：无
返回：List<User>
涉及操作：
  - UserDAO.getAllUsers()
```

## 开发人员 B：图书操作类（BookService）

### 类名：`BookService`

### 负责函数清单：

| 序号 | 函数名 | 功能 | 输入参数 | 返回值 |
|------|--------|------|----------|--------|
| 1 | addNewBook | 添加书籍 | name, author, categoryId, introduction | boolean |
| 2 | removeBook | 删除书籍 | bookId | boolean |
| 3 | modifyBookInfo | 更新书籍信息 | bookId, name, author, categoryId, introduction | boolean |
| 4 | searchBooks | 搜索书籍 | keyword | List<Book> |
| 5 | getBookFullInfo | 获取书籍完整信息 | bookId | Book |
| 6 | borrowBook | 借书 | userId, bookId | boolean |
| 7 | returnBook | 还书 | bookId | boolean |
| 8 | renewBook | 续借书籍 | bookId, days | boolean |
| 9 | addCategory | 添加分类 | categoryName | boolean |
| 10 | removeCategory | 删除分类 | categoryId | boolean |
| 11 | getCategoryWithBooks | 获取分类及书籍 | categoryId | Map |
| 12 | isBookAvailable | 验证书籍可借状态 | bookId | boolean |

### 函数详细说明：

#### 1. addNewBook - 添加书籍
```
功能：添加新书籍
输入：name, author, categoryId, introduction
返回：boolean
涉及操作：
  - BookDAO.addBook()
业务规则：
  - 书名和作者必填
  - 分类ID必须存在（需要调用 BookCategoryDAO.getCategoryById 验证）
  - 默认状态为"可借阅"
```

#### 2. removeBook - 删除书籍
```
功能：删除书籍
输入：bookId (int)
返回：boolean
涉及操作：
  - BookDAO.getBookById()
  - BookDAO.deleteBook()
业务规则：
  - 只能删除"可借阅"状态的书籍
  - 已借出或维护中的书籍不能删除
```

#### 3. modifyBookInfo - 更新书籍信息
```
功能：更新书籍信息
输入：bookId, name, author, categoryId, introduction
返回：boolean
涉及操作：
  - BookDAO.getBookById()
  - BookDAO.updateBook()
业务规则：
  - 书籍必须存在
  - 只能更新"可借阅"或"维护中"状态的书籍
```

#### 4. searchBooks - 搜索书籍
```
功能：按关键词搜索书籍
输入：keyword (String)
返回：List<Book>
涉及操作：
  - BookDAO.searchBooks()
业务规则：
  - 关键词匹配书名或作者
  - 返回所有匹配的书籍
```

#### 5. getBookFullInfo - 获取完整信息
```
功能：获取书籍完整信息（含分类和借阅人）
输入：bookId (int)
返回：Book
涉及操作：
  - BookDAO.getBookDetailById()
```

#### 6. borrowBook - 借书
```
功能：用户借阅书籍
输入：userId (int), bookId (int)
返回：boolean
涉及操作：
  - BookDAO.getBookById() - 验证书籍状态
  - UserDAO.borrowBook() - 更新借阅信息
业务规则：
  - 书籍状态必须为"可借阅"
  - 记录借阅人和借阅时间
  - 状态更新为"已借出"
```

#### 7. returnBook - 还书
```
功能：用户归还书籍
输入：bookId (int)
返回：boolean
涉及操作：
  - BookDAO.getBookById()
  - UserDAO.returnBook()
业务规则：
  - 书籍状态必须为"已借出"
  - 记录归还时间
  - 状态更新为"可借阅"
  - 清空借阅人信息
```

#### 8. renewBook - 续借书籍
```
功能：续借书籍（延长借阅时间）
输入：bookId (int), days (int)
返回：boolean
涉及操作：
  - BookDAO.getBookById()
  - BookDAO.updateBook()
业务规则：
  - 只能续借"已借出"状态的书籍
  - 每次续借不超过30天
```

#### 9. addCategory - 添加分类
```
功能：添加书籍分类
输入：categoryName (String)
返回：boolean
涉及操作：
  - BookCategoryDAO.getCategoryByName()
  - BookCategoryDAO.addCategory()
业务规则：
  - 分类名不能为空且不能重复
```

#### 10. removeCategory - 删除分类
```
功能：删除书籍分类
输入：categoryId (int)
返回：boolean
涉及操作：
  - BookCategoryDAO.getBooksByCategory()
  - BookCategoryDAO.deleteCategory()
业务规则：
  - 分类下有书籍时不能删除
  - 先转移或删除分类下的书籍才能删除分类
```

#### 11. getCategoryWithBooks - 获取分类及书籍
```
功能：获取指定分类及其下的所有书籍
输入：categoryId (int)
返回：Map<String, Object>
涉及操作：
  - BookCategoryDAO.getCategoryById()
  - BookCategoryDAO.getBooksByCategory()
业务规则：
  - 返回分类信息和该分类下的所有书籍
```

#### 12. isBookAvailable - 验证书籍可借状态
```
功能：检查书籍是否可以借阅
输入：bookId (int)
返回：boolean
涉及操作：
  - BookDAO.getBookById()
业务规则：
  - 状态必须为"可借阅"才返回 true
```

## 接口调用依赖关系

```
开发人员 A（UserService）
    │
    ├── 可调用：UserDAO
    └── 可调用：BookService.isBookAvailable()（借书前验证）

开发人员 B（BookService）
    │
    ├── 可调用：BookDAO
    ├── 可调用：BookCategoryDAO
    ├── 可调用：UserDAO
    └── 需配合：UserService.getUserBorrowHistory()（查询借阅历史）
```

---

## 开发规范

1. 每个 Service 类都调用对应的 DAO 层
2. Service 层负责业务逻辑验证
3. DAO 层只负责数据库操作
4. 建议在 Service 层添加日志记录
5. 遇到异常统一打印堆栈并返回 false/null
