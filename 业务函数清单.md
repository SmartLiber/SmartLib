# SmartLib 业务函数清单

## 一、概述

本文档定义图书馆管理系统的业务函数，基于已封装的 DAO 层实现具体业务逻辑。

---

## 二、已封装的属性操作层

| 模块 | DAO | 功能 |
|------|-----|------|
| 书籍 | BookDAO | 书籍的增删改查、关联查询 |
| 用户 | UserDAO | 用户的增删改查、借阅/归还 |
| 分类 | BookCategoryDAO | 分类的增删改查、分类下书籍查询 |

---

## 三、待开发的业务函数清单

### 3.1 用户注册与登录

#### 3.1.1 用户注册

| 项目 | 内容 |
|------|------|
| **函数名** | `registerUser` |
| **功能** | 注册新用户，验证用户名唯一性 |
| **输入参数** | `username` (String), `phone` (String) |
| **返回值** | `boolean` - 注册成功/失败 |
| **涉及操作** | UserDAO.getUserByUsername() - 检查用户名是否存在<br>UserDAO.addUser() - 添加用户 |
| **业务规则** | 用户名不能为空且不能重复<br>手机号格式校验（11位数字） |

#### 3.1.2 用户登录

| 项目 | 内容 |
|------|------|
| **函数名** | `loginUser` |
| **功能** | 用户登录验证 |
| **输入参数** | `username` (String) |
| **返回值** | `User` - 登录成功返回用户对象，失败返回 null |
| **涉及操作** | UserDAO.getUserByUsername() - 查询用户 |
| **业务规则** | 用户名存在即登录成功 |

---

### 3.2 书籍管理

#### 3.2.1 添加书籍

| 项目 | 内容 |
|------|------|
| **函数名** | `addNewBook` |
| **功能** | 添加新书籍 |
| **输入参数** | `name`, `author`, `categoryId`, `introduction` |
| **返回值** | `boolean` - 添加成功/失败 |
| **涉及操作** | BookDAO.addBook() - 添加书籍 |
| **业务规则** | 书名和作者必填<br>分类ID必须存在<br>默认状态为"可借阅" |

#### 3.2.2 删除书籍

| 项目 | 内容 |
|------|------|
| **函数名** | `removeBook` |
| **功能** | 删除书籍 |
| **输入参数** | `bookId` (int) |
| **返回值** | `boolean` - 删除成功/失败 |
| **涉及操作** | BookDAO.getBookById() - 查询书籍<br>BookDAO.deleteBook() - 删除书籍 |
| **业务规则** | 只能删除"可借阅"状态的书籍<br>已借出或维护中的书籍不能删除 |

#### 3.2.3 更新书籍信息

| 项目 | 内容 |
|------|------|
| **函数名** | `modifyBookInfo` |
| **功能** | 更新书籍信息 |
| **输入参数** | `bookId`, `name`, `author`, `categoryId`, `introduction` |
| **返回值** | `boolean` - 更新成功/失败 |
| **涉及操作** | BookDAO.getBookById() - 查询书籍<br>BookDAO.updateBook() - 更新书籍 |
| **业务规则** | 书籍必须存在<br>只能更新"可借阅"或"维护中"状态的书籍 |

#### 3.2.4 搜索书籍

| 项目 | 内容 |
|------|------|
| **函数名** | `searchBooks` |
| **功能** | 按关键词搜索书籍 |
| **输入参数** | `keyword` (String) |
| **返回值** | `List<Book>` - 匹配的书籍列表 |
| **涉及操作** | BookDAO.searchBooks() - 搜索书籍 |
| **业务规则** | 关键词匹配书名或作者<br>返回所有匹配的书籍 |

#### 3.2.5 获取书籍完整信息

| 项目 | 内容 |
|------|------|
| **函数名** | `getBookFullInfo` |
| **功能** | 获取书籍完整信息（含分类和借阅人） |
| **输入参数** | `bookId` (int) |
| **返回值** | `Book` - 包含关联信息的书籍对象 |
| **涉及操作** | BookDAO.getBookDetailById() - 关联查询 |
| **业务规则** | 返回书籍基本信息、分类信息、借阅人信息 |

### 3.3 借阅管理

#### 3.3.1 借书

| 项目 | 内容 |
|------|------|
| **函数名** | `borrowBook` |
| **功能** | 用户借阅书籍 |
| **输入参数** | `userId` (int), `bookId` (int) |
| **返回值** | `boolean` - 借阅成功/失败 |
| **涉及操作** | BookDAO.getBookById() - 查询书籍状态<br>UserDAO.borrowBook() - 更新借阅信息 |
| **业务规则** | 书籍状态必须为"可借阅"<br>记录借阅人和借阅时间<br>状态更新为"已借出" |

#### 3.3.2 还书

| 项目 | 内容 |
|------|------|
| **函数名** | `returnBook` |
| **功能** | 用户归还书籍 |
| **输入参数** | `bookId` (int) |
| **返回值** | `boolean` - 归还成功/失败 |
| **涉及操作** | BookDAO.getBookById() - 查询书籍<br>UserDAO.returnBook() - 更新还书信息 |
| **业务规则** | 书籍状态必须为"已借出"<br>记录归还时间<br>状态更新为"可借阅"<br>清空借阅人信息 |

#### 3.3.3 查询用户借阅记录

| 项目 | 内容 |
|------|------|
| **函数名** | `getUserBorrowHistory` |
| **功能** | 查询用户的借阅历史 |
| **输入参数** | `userId` (int) |
| **返回值** | `List<Book>` - 借阅的书籍列表 |
| **涉及操作** | UserDAO.getBorrowedBooksByUser() - 查询借阅记录 |
| **业务规则** | 返回用户当前借阅的所有书籍 |

#### 3.3.4 续借书籍

| 项目 | 内容 |
|------|------|
| **函数名** | `renewBook` |
| **功能** | 续借书籍（延长借阅时间） |
| **输入参数** | `bookId` (int), `days` (int) - 续借天数 |
| **返回值** | `boolean` - 续借成功/失败 |
| **涉及操作** | BookDAO.getBookById() - 查询书籍<br>BookDAO.updateBook() - 更新借阅信息 |
| **业务规则** | 只能续借自己借的书籍<br>只能续借"已借出"状态的书籍<br>每次续借不超过30天 |

### 3.4 分类管理

#### 3.4.1 添加分类

| 项目 | 内容 |
|------|------|
| **函数名** | `addCategory` |
| **功能** | 添加书籍分类 |
| **输入参数** | `categoryName` (String) |
| **返回值** | `boolean` - 添加成功/失败 |
| **涉及操作** | BookCategoryDAO.getCategoryByName() - 检查分类是否存在<br>BookCategoryDAO.addCategory() - 添加分类 |
| **业务规则** | 分类名不能为空且不能重复 |

#### 3.4.2 删除分类

| 项目 | 内容 |
|------|------|
| **函数名** | `removeCategory` |
| **功能** | 删除书籍分类 |
| **输入参数** | `categoryId` (int) |
| **返回值** | `boolean` - 删除成功/失败 |
| **涉及操作** | BookCategoryDAO.getBooksByCategory() - 检查分类下是否有书籍<br>BookCategoryDAO.deleteCategory() - 删除分类 |
| **业务规则** | 分类下有书籍时不能删除<br>先转移或删除分类下的书籍才能删除分类 |

#### 3.4.3 获取分类及书籍

| 项目 | 内容 |
|------|------|
| **函数名** | `getCategoryWithBooks` |
| **功能** | 获取指定分类及其下的所有书籍 |
| **输入参数** | `categoryId` (int) |
| **返回值** | `Map<String, Object>` - 包含分类信息和书籍列表 |
| **涉及操作** | BookCategoryDAO.getCategoryById() - 查询分类<br>BookCategoryDAO.getBooksByCategory() - 查询分类下书籍 |
| **业务规则** | 返回分类信息和该分类下的所有书籍 |
